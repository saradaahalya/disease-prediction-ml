# Disease Prediction from Gene Expression Data

## Overview
This project uses machine learning to predict diseases based on gene expression data.

## Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

## Workflow
1. Data preprocessing
2. Feature selection
3. Model training
4. Evaluation

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
